title: "Retrieving the Hardware ID for an Appliance"

---
To retrieve the license key for an appliance from the Support portal, you need its hardware ID (serial number). After setting up or pre-provisioning a physical or vNIOS virtual appliance, obtain the hardware ID or the virtual host hardware ID for a vNIOS virtual appliance.

A physical appliance's hardware ID is a 16-digit number and a virtual appliance's ID is a 32-digit uppercase alphanumeric string.

1. Sign in to the NIOS CLI.  
   Or, open a terminal session for the VM and launch the NIOS CLI.
2. Use one of the following commands to obtain the hardware ID, and then copy ID value for convenience:
   
   - Run the `show hwid` command as shown in the following example:  
     `Infoblox &gt; show hwid`  
     `  Hardware ID: 564Dxxxx633D31xxxxF7684AE2xxxxE7`
   - Run the `show license` command as shown in the following example:  
     `Infoblox &gt; show license`  
     ` Version : 9.1.0-54505-4f8c7xxxxcc5`   
     ` Hardware ID : 564Dxxxx633D31xxxxF7684AE2xxxxE7`

> **warning**
>
> **Note**
> 
> Make a note of the hardware IDs that you obtain during this procedure. Each of these unique Hardware ID values can be associated with a License Activation ID from your Contract Notification email. If a license key is installed for the current VM, that key value also appears in the output for the `show license` command.
