You can create a new DNS Infrastructure Protection profile by cloning an existing one. The appliance creates a new profile with the settings that are copied from the source profile. You can also associate extensible attributes with the profile. Note that members in the source profile are not carried over to the cloned profile. You must associate new members with the newly cloned profile.  
To clone a DNS Infrastructure Protection profile:

1. From the **Data** **Management** tab, select the **Security** tab -&gt; **Profiles** tab, select the DNS Infrastructure Protection profile that you want to clone, click the Action icon  and select **Clone**.
2. In the *Clone* *DNS Infrastructure* *Protection* *Profile* *Wizard*, enter a name for the DNS Infrastructure Protection profile and modify the details. For more information, see *[Adding DNS Infrastructure Protection Profiles](/nios90/by-id/280668792)*.
3. Associate extensible attributes and members with the DNS Infrastructure Protection profile.
4. Save the configuration.
