The following icons appear in the **Grid** tab:

| **Icon** | **Icon** **Name** | **Description** | **Tab/Table/Panel** |
| --- | --- | --- | --- |
|    | Backup | Backs up the configuration file and database | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; Toolbar |
|    | Restore | Restores the configuration file and database | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; Toolbar |
|    | bloxTools | Performs bloxTools services | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; Toolbar |
|    | Certificate | Creates, generates, uploads, or downloads an HTTPS certificate | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; **Members **tab -&gt; *member *-&gt; Toolbar |
|    | Control | Restarts, reboots, or shuts down a member | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; **Members **tab -&gt; *member *-&gt; Toolbar |
|    | Manage Services | Manages member services | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; **Members **tab -&gt; *member* |
|    | Syslog | Displays the syslog file | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; **Members **tab -&gt; *member* -&gt; Toolbar |
|    | Traffic Capture | Captures the traffic report on a member | **Infoblox Grid** tab-&gt; **Grid** **Manager** tab -&gt; Members tab -&gt; *member* -&gt; Toolbar |
