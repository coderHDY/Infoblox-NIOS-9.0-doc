To view the client certificates that were generated:

1. Grid: From the **Infoblox** **Grid** tab, select the **Grid** **Manager** tab.  
   Grid Master Candidate: From the **Infoblox** **Grid** tab, select the** Grid** **Manager** tab -&gt; **Members** tab -&gt; *member* checkbox.
2. From the Toolbar, click **Certificates** -&gt; **Client** **Cert** -&gt; **View** **Client** **Certificate**, and select either **RSASHA1** or **RSASHA256.**

The appliance displays the selected certificate.
