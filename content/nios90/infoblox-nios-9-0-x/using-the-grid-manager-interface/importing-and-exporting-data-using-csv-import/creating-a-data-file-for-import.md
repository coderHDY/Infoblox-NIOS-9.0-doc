If you are migrating new data into the database, you must prepare the data file using the correct format and syntax before you can import it successfully. You must include all the required fields and understand the dependencies among some of the fields.

> **info**
>
> Starting in NIOS 9.0.x, you can import and export **SVCB (TYPE64)** and **HTTPS (TYPE65)** DNS records using CSV.

For detailed information about the guidelines, supported record types, and interdependencies among fields, refer to the [*Infoblox*](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference)[ ](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference)[*CSV*](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference)[ ](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference)[*Import*](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference)[ ](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference)[*Reference*](/nios90/infoblox-nios-9-0-x/reference-information/csv-import-reference).
