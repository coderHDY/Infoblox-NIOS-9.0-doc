To modify the information you entered for an existing URL link:

1. In the *Finder* panel, expand **URLLinks**.
2. Hover your mouse over the URL you want to modify, and then click the Edit icon.
3. In the *URLConfiguration* dialog box, modify the information as described in [*Adding URL Links*](/nios90/infoblox-nios-9-0-x/using-the-grid-manager-interface/performing-basic-operations-using-grid-manager/finding-and-restoring-data/managing-third-party-url-links/adding-url-links).
