To add URL links:

1. In the *Finder* panel, expand **URLLinks**.
2. Click **Add**.
3. In the *URL Configuration* dialog box, complete the following:
   
   - **URL**: Enter the URL of the destinati characters. When you enter the URL, the appliance validates the entry. You cannot save the entry if the URL is not in a valid format.
   - **Name**: Enter a name that represents the portal or site of the URL.
   - **Setasglobalparameter**: This field appears only if you log in as a superuser. Select this checkbox to make the URL link globally available to all users.
   - **Logo**: Click **Upload** to add a logo to the URL. The appliance displays the logo in 16x16 pixels. Click **ResettoDefault** to use the default logo.
4. Save the configuration.
