The **Top** **10** **Grid** **Members** tab displays a stacked bar chart that tracks the top Grid members with the most total counts of DNS Infrastructure Protection events. Each severity level is represented with a different color. The report displays the top 10 members in descending order.  
If you have configured a reporting member in the Grid, the **Go** **To** **History** link is displayed in this tab. You can click **Go** **To** **History** to view the *DNS Infrastructure* *Protection* *Event* *Count* *By* *Member/Member* *Group* *Trend* report in the **Reporting** tab. To navigate back to the Security dashboard from the **Reporting** tab, click **Back** **to** **Security** **Dashboard** at the top left corner of the navigation bar in the **Reporting** tab.
