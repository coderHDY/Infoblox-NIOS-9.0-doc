After you create a dashboard template, you can assign it to an admin group. Admin users in this admin group can access the tasks you define in the template.  
To assign a dashboard template to an admin group, see [*About Admin Groups*](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-administrators/about-admin-groups).
