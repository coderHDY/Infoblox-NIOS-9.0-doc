If you belong to an approver admin group, you can view, approve, or reject tasks that are pending your approval in the **Task** **Manager** tab. For information, see [*Viewing Tasks*](/nios90/infoblox-nios-9-0-x/using-the-grid-manager-interface/scheduling-tasks/viewing-tasks). Submitter's can view all pending and completed tasks they have submitted.
