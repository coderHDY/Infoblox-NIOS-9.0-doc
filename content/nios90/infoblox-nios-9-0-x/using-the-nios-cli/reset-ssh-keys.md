The `reset`` ``ssh_keys` command resets the SSH keys of the system.

# Syntax

`reset ssh_keys`

This command has no arguments.

# Example

`Infoblox &gt; ``reset`` ``ssh_keys`  
` The system's SSH keys were reset.`
