The  `set max_recursion_depth` command sets the limit on the allowed number of levels of recursion named.

# Syntax

`set max_recursion_depth &lt;value&gt;`

| **Argument ** | **Description** |
| --- | --- |
| `value` | The depth value in the range 1 - 100. Default depth value is 7. |

# Examples

Infoblox &gt; `set max_recursion_depth`
