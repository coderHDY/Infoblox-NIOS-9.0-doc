Use the `set ha_link_check `CLI command to configure HA link checking. The value of the command is set to `off` by default.

# Syntax

`set ha_link_check [on | off]`

| **Argument** | **Description** |
| --- | --- |
| `on` | Turns on HA link checking. |
| `off` | Turns off HA link checking. |
