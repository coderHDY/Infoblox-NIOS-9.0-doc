The `set ms_sticky_ip` command enables/disables ms_sticky_ip.

# Syntax

`set ms_sticky_ip [on|off]`

| **Argument ** | **Description** |
| --- | --- |
| on | Enables ms_sticky_ip. |
| off | Disables ms_sticky_ip. |
