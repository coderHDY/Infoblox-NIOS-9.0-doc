The `set cloud_services_portal_force_refresh` command set the flag to request all domains detected by Infoblox Threat Insight engine in the Cloud.

.

# Syntax

`set cloud_services_portal_force_refresh`

There are no arguments for this command.

# Examples

Infoblox &gt; `set cloud_services_portal_force_refresh`  
                `Do you want to proceed? (y or n):n`
