The `set`` ``nosafemode`` `command disables safe mode on the NIOS appliance by re-enabling DNS and DHCP services. For more information, see *[set safemode](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-safemode)*.

# Syntax

`set nosafemode`

This command has no arguments.

# Example

`Infoblox &gt; ``set`` ``nosafemode`
