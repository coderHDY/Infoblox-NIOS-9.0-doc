The `set vdca-tcp-mode` CLI command, introduced in NIOS 9.0.4,  enables or disables DNS over TCP service at vdca-tcp-mode.

# Syntax

`set vdca-tcp-mode &lt;on|off&gt;`

| **Argument** | **Description** |
| --- | --- |
| on | Enables DNS over TCP service |
| off | Disables DNS over TCP service |

# Examples

## Enable DNS over TCP service

`Infoblox &gt; set vdca-tcp-mode on `

## Disable DNS over TCP service

`Infoblox &gt; set vdca-tcp-mode off `
