The `set ocsp`** off **command disables OCSP authentication service configuration and allow you to login without OCSP validation. It also terminate administrative sessions for users who are currently logged in.

# Syntax

`set ocsp off`

There are no arguments for this command.

# Example

`Infoblox &gt; ``set ocsp off`
