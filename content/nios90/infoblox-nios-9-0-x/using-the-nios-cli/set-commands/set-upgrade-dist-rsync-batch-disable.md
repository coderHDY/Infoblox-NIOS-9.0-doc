The set upgrade_dist rsync_batch disable command disables the resync-batch mode for all members on the Grid.



# Syntax

`set upgrade_dist rsync_batch disable`

# Examples

`Infoblox &gt;`` ``set upgrade_dist rsync_batch disable`
