The **delete** command deletes a specific file or a set of files.

# Syntax

`delete [file``]`

| **Argument** | **Description** |
| --- | --- |
| *file* | The name of the file which needs to be deleted |

# Example

Infoblox &gt; **delete** **abc.csv**
