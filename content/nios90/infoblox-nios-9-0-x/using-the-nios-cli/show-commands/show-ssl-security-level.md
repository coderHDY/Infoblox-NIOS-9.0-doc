The `show ssl_security_level` command is used to display the current OpenSSL security level set. For more information about setting OpenSSL security level, see [*set ssl security level*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-ssl-security-level).

# Syntax

Infoblox &gt; `show ssl_security_level`

# Example

Infoblox &gt; `show ssl_security_level`
`SSL Security Level: 1`
