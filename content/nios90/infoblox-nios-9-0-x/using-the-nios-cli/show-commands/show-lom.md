The `show lom` command displays the LOM (Lights Out Management) settings for the IPMI interface. To configure the network settings for the IPMI interface, use the [*set lom*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-lom) command.

# Syntax

show lom

This command has no arguments.

# Example

`Infoblox &gt; show lom`  
` LOM for Grid: enabled`  
` LOM for member: enabled (inherit) `

`Network settings:`  
` IP Address: 192.0.2.10`  
` Subnet Mask: 255.255.255.0 Default Gateway IP: 192.0.2.10 `
