You can use the `show rabbitmq_memory_watermark` command to show RabbitMQ memory watermark.

# Syntax

`show rabbitmq_memory_watermark`

# Example

`Infoblox &gt; show rabbitmq_memory_watermark`

`Rabbit memory watermark : 10`
