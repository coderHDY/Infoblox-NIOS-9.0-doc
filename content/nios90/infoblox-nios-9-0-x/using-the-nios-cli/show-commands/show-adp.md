The `show adp` command displays DNS Infrastructure Protection details on the supported platform. You can use this command only if **Threat** **Protection** (hardware based) or **Threat** **Protection** **(Software** **add-on)** licenses are installed on the platform.

# Syntax

`show adp`

This command has no arguments.

# Example

`Infoblox &gt; show adp`  
` Threat Protection:                 Enabled `  
`Threat Protection monitor mode:    Disabled`  
` Threat Protection event stats:     CRITICAL=0 MAJOR=0 WARNING=0 INFORMATIONAL=2 `  
`Log level:                        6(Info) `
