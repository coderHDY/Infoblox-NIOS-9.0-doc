You can use `show hw_ident`CLI command to display hardware identification LED status.

# Syntax

`show hw_ident [ on | off ]`

| **Argument** | **Description** |
| --- | --- |
| `on` | hwident : on |
| `off` | hwident : off |

# Example

`Infoblox &gt; show hw_ident on`
