The `show perl_mod_access` CLI command displays whether access to the Perl module in NIOS is enabled or disabled. For more information to set the access to the Perl module, see [set perl_mod_access](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-perl-mod-access).

# Syntax

`show perl_mod_access`

# Example

`Infoblox &gt; show perl_mod_access`

`Access to the Perl module is disabled.`  
`Infoblox &gt;`
