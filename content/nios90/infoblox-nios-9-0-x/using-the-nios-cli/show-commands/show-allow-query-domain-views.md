The `show allow_query_domain_views `command displays the list of all DNS views that have the allow query domain ACLs configured. For information about adding, updating, or deleting the allow query domain ACLs, see *[set allow_query_domain](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-allow-query-domain)*. To view the list of all domains that have allow query domain ACLs, see *[show allow_query_domain](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/show-commands/show-allow-query-domain)*.

# Syntax

`show allow_query_domain_views`

This command has no arguments.

# Example

`Infoblox &gt;`` ``show allow_query_domain_views`

`DNS views that have domain ACL associations:`  
`1. default`  
`2. dns_view`
