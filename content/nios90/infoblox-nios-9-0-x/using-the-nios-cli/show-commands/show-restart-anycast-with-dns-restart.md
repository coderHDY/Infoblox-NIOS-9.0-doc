The` ``show restart_anycast_with_dns_restart` command displays the status of the `set restart_anycast_with_dns_restart` command. For information about how to set restart anycast command, see *[set restart_anycast_with_dns_restart](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-restart-anycast-with-dns-restart)* .

# Syntax

`show restart_anycast_with_dns_restart`

This command has no arguments.

# Example

The following is an example of the output displayed when you execute the `show restart_anycast_with_dns_restart` command:

`Infoblox &gt; ``show restart_anycast_with_dns_restart`` `

`restart_anycast_with_dns_restart is set to "on" (default value). `

`Therefore, when the DNS service restarts, the anycast service also restarts.`
