If you have configured Infoblox Subscriber Services, use the `set subscriber_secure_data list_ip_space_desc` command to display the list of configured IP space discriminators.

# Syntax

`show subscriber_secure_data list_ip_space_desc`

This command has no arguments.

# Examples

`Infoblox &gt; show subscriber_secure_data list_ip_space_desc`  
`NAS-IP-Address=10.2.1.1`  
`NAS-IP-Address=10.36.1.10`  
`NAS-IP-Address=10.120.252.24`  
`NAS-IP-Address=10.36.120.10`
