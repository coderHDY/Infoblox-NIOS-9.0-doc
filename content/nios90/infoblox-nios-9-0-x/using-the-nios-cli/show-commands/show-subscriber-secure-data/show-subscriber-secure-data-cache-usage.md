If you have configured Infoblox Subscriber Services, use the `set subscriber_secure_data cache_usage` command to display the count of subscriber entries in cache.

# Syntax

`show subscriber_secure_data dca_cache_usage`

This command has no arguments.

# Examples

`Infoblox &gt; show subscriber_secure_data cache_usage`  
`122798 accounting records in the cache.`
