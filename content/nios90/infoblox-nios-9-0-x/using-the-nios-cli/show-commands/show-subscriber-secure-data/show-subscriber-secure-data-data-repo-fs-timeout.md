If you have configured Infoblox Subscriber Services, use the `set subscriber_secure_data data_repo_fs_timeout` command to display the fast sync timeout value of the data repository.

## Syntax

`show subscriber_secure_data data_repo_fs_timeout`

This command has no arguments.

## Example

`Infoblox &gt; show subscriber_secure_data data_repo_fs_timeout`  
`Data Repo fast sync timeout is 30 minutes.`
