If you have configured Infoblox Subscriber Services, use the `set subscriber_secure_data dca_abl_record_count` command to display the number of ABL and non ABL subscriber records in DNS Cache Acceleration.

## Syntax

`show subscriber_secure_data dca_abl_record_count`

This command has no arguments.

## Example

`Infoblox &gt; set subscriber_secure_data dca_abl_record_count`  
`Non ABL subscribers count : 4644`
