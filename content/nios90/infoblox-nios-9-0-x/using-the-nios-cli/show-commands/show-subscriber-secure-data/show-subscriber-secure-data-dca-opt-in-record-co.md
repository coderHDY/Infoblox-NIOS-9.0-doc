If you have configured Infoblox Subscriber Services, use the `set subscriber_secure_data dca_opt_in_record_count` command to display the number of opt in subscriber records in DNS Cache Acceleration.

## Syntax

`show subscriber_secure_data dca_opt_in_record_count`

This command has no arguments.

## Example

`Infoblox &gt; show subscriber_secure_data dca_opt_in_record_count`  
`Subscriber list count : 10`
