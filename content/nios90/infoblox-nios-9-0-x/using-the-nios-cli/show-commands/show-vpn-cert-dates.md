Use the `show`` ``vpn_cert_dates` command to display the start and end dates of the Infoblox appliance certificate. This information is also included in the Support Bundle.

# Syntax

`show vpn_cert_dates`

This command has no arguments.

# Example

The following is an example of the command:

`Infoblox &gt; show vpn_cert_dates `  
`Start Date=Dec 13 11:00:00 2003 GMT`  
` End Date=May 20 11:00:00 2019 GMT`
