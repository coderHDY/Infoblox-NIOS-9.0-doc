The `show doh-status` command displays the status of the DNS over HTTPS service. Use this command to verify whether the feature is enabled. For steps to enable DNS over TLS, see *[Configuring DNS over HTTPS](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dns-services/configuring-dns-over-tls-and-dns-over-https-serv)*.

# Syntax

`show doh-status`

This command has no arguments.

# Example

`Infoblox &gt; ``show doh-status`  
`DoH is enabled`  
`DoH trace is off`  
`DoH key logging is off`  
`Max server sockets: 128`  
`curr server sockets: 2`  
`curr client sockets: 0`
