The `show ms_sticky_ip` command displays whether the ms_sticky_ip is turned on or off.

# Syntax

`show ms_sticky_ip`

This command has no arguments.

# Example

`Infoblox &gt; show ms_sticky_ip`

`show ms_sticky_ip`  
`ms_sticky_ip is off`
