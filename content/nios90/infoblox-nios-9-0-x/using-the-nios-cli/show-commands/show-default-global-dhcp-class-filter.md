The `show default_global_dhcp_class_filter` command displays the current default value of the **Apply this filter as a global DHCP class** checkbox in the *Add IPv4 Option Filter Wizard* and *Add IPv6 Option Filter Wizard *screens.

For information about the **Apply this filter as a global DHCP class** checkbox, refer to [Defining Option Filters](/nios90/infoblox-nios-9-0-x/administering-nios/dhcp/configuring-dhcp-filters/configuring-option-filters/defining-option-filters).

## Syntax

`show default_global_dhcp_class_filter`

This command has no arguments.

## Example

`Infoblox &gt; show default_global_dhcp_class_filter`  
`The default_global DHCP class filter checkbox is set to 'enabled'`



For more information on enabling or disabling the **Apply this filter as a global DHCP class** checkbox, refer to the [set command](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-default-global-dhcp-class-filter).
