The `show dns-over-tls-status` command displays the status of DNS over TLS service. Use this command to verify whether the feature is enabled. For steps to enable DNS over TLS, see *[Configuring DNS over TLS](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dns-services/configuring-dns-over-tls-and-dns-over-https-serv)*.

# Syntax

`show dns-over-tls-status`

This command has no arguments.

# Example

`Infoblox &gt; ``show dns-over-tls-status`  
`DoT is enabled`  
`DoT trace is off`  
`DoT key logging is off`  
`Max server sockets: 128`  
`curr server sockets: 4`  
`curr client sockets: 0`
