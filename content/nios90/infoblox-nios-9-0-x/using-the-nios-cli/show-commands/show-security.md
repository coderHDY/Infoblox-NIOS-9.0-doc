The `show`` ``security` command shows the current security settings and whether access to the NIOS appliance through the GUI is restricted. For more information, see *[set security](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-security)*.

# Syntax

`show security`

This command has no arguments.

# Example

`Infoblox &gt; ``show`` ``security`  
` current security settings: `  
` Access restricted: false`
