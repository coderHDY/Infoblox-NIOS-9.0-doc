The `show adp_bfd_bypass` command displays the current status of DNS Infrastructure Protection BFD bypass, specifying whether it is enabled or disabled. You can use this command only if **Threat** **Protection** (hardware based) or **Threat** **Protection** **(Software** **add-on)** licenses are installed on the platform.   
For information about how to set the adp_bfd_bypass, see [*set adp_bfd_bypass*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-adp-bfd-bypass).

# Syntax

`show adp_bfd_bypass`

This command has no arguments.

# Example

`Infoblox &gt; show adp_bfd_bypass`
