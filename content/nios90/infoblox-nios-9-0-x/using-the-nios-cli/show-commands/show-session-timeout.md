The `show`` ``session_timeout`` `command shows how long a session remains open when there is no user activity. For more information, see *[set session_timeout](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-session-timeout)*.

# Syntax

`show session_timeout`

This command has no arguments.

# Example

`Infoblox &gt; ``show`` ``session_timeout`  
` Current GUI/CLI timeout is 31536000 seconds (8760:00:00)`
