The `show query_logging_warnings` command displays the current status of query logging warnings. This command is available starting from NIOS 9.0.7.

For information on how to enable or disable query logging warnings, see* *[*set query_logging_warnings*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-query-logging-warnings)*.*

# Syntax

`Infoblox &gt; show query_logging_warnings`

This command has no arguments.

# Example

`Infoblox &gt; show query_logging_warnings  `  
`DNS query logging warnings or response logging warnings have been disabled.`



`Infoblox &gt; show query_logging_warnings  `  
`Either DNS query logging warnings or response logging warnings have been enabled or the default values are being used.`
