You can use `show logfiles`command to display the size of the system log files.

# Syntax

`show logfiles`

This command has no arguments.

# Example

`Infoblox &gt; show logfiles`

`Logfiles present on the system`  
`--------------------------------`  
`audit.log                        146 k`  
`infoblox.log                    3128 k`  
`syslog                          1770 k`
