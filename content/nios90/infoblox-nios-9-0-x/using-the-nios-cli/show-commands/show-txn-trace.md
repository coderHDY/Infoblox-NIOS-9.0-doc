The `show txn_trace` command enables/disables tracing of DB transaction log.

# Syntax

`show txn_trace [ on | off ]`

| **Argument** | **Description** |
| --- | --- |
| on | Enable tracing for DB transactions |
| off | Disable tracing for DB transactions |

# Example

## Enables tracing for DB transactions

`Infoblox &gt; show txn_trace on`

## Disables tracing for DB transactions

`Infoblox &gt; show txn_trace off`
