The` show ssl_tls_settings` command shows whether SSL/TLS settings are set to the default or override mode. You can use the `set ssl_tls_settings` command in override mode to change the SSL/TLS settings. For information, see [*set ssl_tls_settings*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-ssl-tls-settings).

# Syntax

`show ssl_tls_settings`

This command has no arguments.

# Example

`Infoblox &gt; show ssl_tls_settings`  
`SSL/TLS settings: default.`  
`Use 'ssl_tls_protocols' and 'ssl_tls_ciphers' to see current settings.`
