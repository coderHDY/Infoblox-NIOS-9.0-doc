The `show` **fips**`_mode` command displays settings of the FIPS mode.

# Syntax

`show fips_mode`

This command has no arguments.

# Example

`Infoblox &gt; ``show`` ``fips_mode`  
` FIPS Mode Setting:`  
` FIPS Mode Enabled (grid-level): Yes `
