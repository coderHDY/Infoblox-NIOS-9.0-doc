The **show** **remote_console** command displays remote console access settings. This command allows you to see if remote console access is enabled without accessing the GUI. You can set this option at the member level and the Grid level. For information on how to change your remote_console, see *[set remote_console](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-remote-console)*.

# Syntax

`show remote_console`

This command has no arguments.

# Example

`Infoblox &gt; ``show`` ``remote_console`  
` current remote console access settings:`  
` Remote console access enabled (Grid-level): true`
