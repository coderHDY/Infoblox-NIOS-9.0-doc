The `show`` ``dhcpd_recv_sock_buf_size` displays the current DHCP receive socket buffer size. The default is 1024 kilobytes. For information about how to set the receive socket buffer size, see *[set dhcpd_recv_sock_buf_size](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-dhcpd-recv-sock-buf-size)*.

# Syntax

`show dhcpd_recv_sock_buf_size`

This command has no arguments.

# Examples

**For** **NIOS** **Appliances**

`Infoblox &gt; ``show`` ``dhcpd_recv_sock_buf_size`  
` DHCP 'dhcpd' UDP receive socket buffer size: 1500 KB`
