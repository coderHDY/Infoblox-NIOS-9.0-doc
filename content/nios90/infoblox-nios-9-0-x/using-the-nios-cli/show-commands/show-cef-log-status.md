The `show cef_log_status ` command displays the CEF log status for violations at both BIND and virtual DNS Acceleration. For more information on how to enable or disable the CEF logging, see [*set cef_logging*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-cef-logging).

# Syntax

`show cef_log_status`

This command has no arguments.

# Example

`Infoblox &gt; show cef_log_status `

`BIND CEF logging: on`

`vDCA CEF logging: on`
