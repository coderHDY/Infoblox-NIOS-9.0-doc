The `show`` ``ipv6_disable_on_dad` command displays whether IPv6 is disabled, when a duplicate IPv6 address is detected, on the corresponding interface.

# Syntax

`show ipv6_disable_on_dad`

# Examples

`Infoblox &gt; ``show`` ``ipv6_disable_on_dad`  
` Disable IPv6 if duplicate IPv6 address detected: off `  
` `

`Infoblox &gt; ``show`` ``ipv6_disable_on_dad`  
` Disable IPv6 if duplicate IPv6 address detected: on`
