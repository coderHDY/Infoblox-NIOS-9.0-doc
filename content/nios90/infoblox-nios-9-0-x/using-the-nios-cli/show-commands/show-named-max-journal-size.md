The command `show named_max_journal_size` is used to show the max journal size limit.

# Syntax

`show named_max_journal_size &lt;grid value&gt;`

| **Argument** | **Description** |
| --- | --- |
| Grid value | The max grid value can be up-to 100000. |

# Example

`Infoblox &gt; show named_max_journal_size 5000`
