The `show send_grid_info` CLI command displays whether sharing Grid-related information to Infoblox is currently enabled or disabled.

> **warning**
>
> **Note**:
> 
> This feature is exclusively accessible to superusers, can only be activated via CLI, and is executable only on the Grid Master.

# Syntax

`show send_grid_info`

# Example

`Infoblox &gt; show send_grid_info`  
Grid level information sharing with Infoblox : enabled
