You can use`show dca_incoming_deatiled_packet_stats`CLI command to display incoming detailed packet statistics at vDCA.

# Syntax

`show dca_incoming_deatiled_packet_stats`

# Example

`Infoblox &gt; show dca_incoming_detailed_packet_stats`

`Non EDNS0 queries: 0`  
`EDNS0 queries with local ID: 0`  
`EDNS0 queries with policy ID: 0`  
`EDNS0 queries with other options: 0`  
`Queries from subscriber: 0`  
`Queries from policy ID subscriber: 0`  
`Queries from IPv4 prefix subscriber: 0`  
`Queries from IPv6 prefix subscriber: 0`  
`DCA PCP violation hit: 0`  
`DCA WPCP violation hit: 0`
