The `show`` ``log_txn_id`` `command displays whether DHCP transaction ID logging is on or off. By default, DHCP transaction ID logging is enabled. Use the [*set*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-log-txn-id)[ ](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-log-txn-id)[*log_txn_id*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-log-txn-id) to enable or disable logging of DHCP transaction IDs.

# Syntax

`show log_txn_id`

This command has no arguments.

# Example

`Infoblox &gt; ``show`` ``log_txn_id`  
` DHCP Transaction id logging turned OFF`
