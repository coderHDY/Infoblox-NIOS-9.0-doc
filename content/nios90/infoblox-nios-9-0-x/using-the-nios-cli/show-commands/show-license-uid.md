The `show license_uid` command displays the license UID of the Grid. The UID is required when requesting  
 Grid-wide licenses. The UID that the appliance returns is the same as the **License** **Pool** **Container** **UID** that is used for obtaining dynamic licenses.

# Syntax

`show license_uid`

This command has no arguments.

# Examples

`Infoblox &gt; ``show`` ``license_uid`  
` The grid-wide license unique ID (same as LPC_UID): e51f90527dce4708bc1ada576286d26a`
