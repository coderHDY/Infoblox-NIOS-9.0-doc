The `show support_timeout` command displays the custom timeout value set for the support bundle download. For information about how to set the timeout value, see [*set support_timeout*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-support-timeout).

# Syntax

`show support_timeout`

This command has no arguments.

# Example

`Infoblox &gt; show support_timeout `

`Support timeout is 2600`
