The `show allow_grid_token` command displays whether a Grid member is allowed to use a grid token required to join the Infoblox portal. For more information, see [*set allow_grid_token*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-allow-grid-token).

# Syntax

`show allow_grid_token`

This command has no arguments.

# Example

`Infoblox &gt; show allow_grid_token`  
`grid token is on`
