The `show` `cc_mode` command displays the Common Criteria settings.

# Syntax

`show cc_mode`

This command has no arguments.

# Example

`Infoblox &gt; ``show`` ``cc_mode`  
` Common Criteria Mode Setting:`  
` Common Criteria Mode Enabled (grid-level): Yes `
