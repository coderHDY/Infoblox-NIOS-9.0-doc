A user who belongs to the SAML Admin role has the following permissions set:

| **Permission Type** | **Resource Type** | **Permission** |
| --- | --- | --- |
| SAML Permissions | SAML Admin group | RW |
| SAML Permissions | SAML Authentication Services | RW |

For information about SAML authentication, see *[Authenticating Admins Using SAML](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-administrators/authenticating-admins-using-saml)*.
