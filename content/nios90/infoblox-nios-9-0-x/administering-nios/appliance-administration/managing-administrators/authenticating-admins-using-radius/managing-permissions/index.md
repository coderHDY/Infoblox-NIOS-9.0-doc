After you define permissions for an admin group and role, you can do the following:

- View the permissions, as described in [*Viewing Permissions*](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-administrators/authenticating-admins-using-radius/managing-permissions/viewing-permissions).
- Filter a list of permissions, as described in [*Filtering the List of Permissions*](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-administrators/authenticating-admins-using-radius/managing-permissions/filtering-the-list-of-permissions).
- Modify the permissions, as described in [*Modifying Permissions*](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-administrators/authenticating-admins-using-radius/managing-permissions/modifying-permissions).
- Delete the permission, as described in [*Deleting Permissions*](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-administrators/authenticating-admins-using-radius/managing-permissions/deleting-permissions).
