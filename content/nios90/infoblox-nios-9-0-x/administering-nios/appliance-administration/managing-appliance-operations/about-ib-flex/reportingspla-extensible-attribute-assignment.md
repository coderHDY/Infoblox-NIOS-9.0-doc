According to the member type and the license installed on a reporting member, the extensible attribute **ReportingSPLA** is assigned. The values assigned for the extensible attribute are as follows:

- If a Grid Master has a Flex Grid Activation license installed, it can be assigned with one of the following extensible attribute values:
  
  - **IB-FLEX**:: For an IB-FLEX member
  - **HW/SW**: For a non IB-FLEX member
- If a Grid Master has a Flex Grid Activation for Managed Services license installed, it can be assigned with one of the following extensible attribute values:
  
  - **Managed IB-FLEX:**: For an IB-FLEX member
  - **Managed HW/SW**: For a non IB-FLEX member

Data in the following reporting dashboards can be filtered for individual members based on the extensible attribute ReportingSPLA:

- Managed DDI features enabled
- SPLA Grid Licensing Features Enabled
- Managed DDI Peak IP Usage Trend
- Managed DNS Peak Usage Trend
- DNS Effective Peak Usage Trend for SPLA Grid License
