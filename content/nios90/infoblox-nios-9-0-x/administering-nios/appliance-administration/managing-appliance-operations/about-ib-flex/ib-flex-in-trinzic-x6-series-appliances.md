title: "IB-FLEX in Trinzic X6 Series Appliances"
source: "/space/nios90/1432754730"
pageId: "1432754730"
---
All of the Trinzic X6 series appliances: TE-1506, TE-1606, TE-2306 and TE-4106 can be used as IB-FLEX members or Grid Masters if the hardware type is set to IB-FLEX using the CLI command [*set hardware-type*](/nios90/infoblox-nios-9-0-x/using-the-nios-cli/set-commands/set-hardware-type).

The DNS Cache Acceleration service runs on all Trinzic X6 Series appliances when used as IB-FLEX. Infoblox supports DNS Cache Acceleration only on TE-2306 and TE-4106 appliances. Although, the DNS Cache Acceleration can be enabled on TE-1506, and TE-1606 appliances, Infoblox does not support it.
