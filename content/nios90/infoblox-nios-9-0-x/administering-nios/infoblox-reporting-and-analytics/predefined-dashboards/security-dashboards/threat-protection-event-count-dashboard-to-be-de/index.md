> **warning**

> **Note**
> 
> This dashboard will be deprecated from NIOS 9.1.0 onwards. For future reference, use [*DNS Infrastructure Protection Event Count Dashboard*](/nios90/infoblox-nios-9-0-x/administering-nios/infoblox-reporting-and-analytics/predefined-dashboards/security-dashboards/dns-infrastructure-protection-event-count-dashbo).

The *Threat Protection Event Count Dashboard* is a predefined custom dashboard comprising sub-dashboards listed in the table below. Duplicates of these sub-dashboards exist under new names, replacing "Threat Protection" with "DNS Infrastructure Protection" since NIOS 9.0.8. Dashboards with the previous names will be deprecated from 9.1.0 onwards.
