After you have set up reporting members and defined clustering type, you can monitor the cluster status through the following:

- View the reporting member service status, as described in Monitoring Grid Services, see [*Monitoring Services*](/nios90/infoblox-nios-9-0-x/administering-nios/monitoring/monitoring-the-appliance/monitoring-services).
- Check license usage by the reporting member, as described in [*Home Dashboards*](/nios90/infoblox-nios-9-0-x/administering-nios/infoblox-reporting-and-analytics/home-dashboards).
