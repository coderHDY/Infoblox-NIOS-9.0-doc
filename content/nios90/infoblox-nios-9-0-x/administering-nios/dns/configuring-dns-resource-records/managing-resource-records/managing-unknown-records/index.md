title: "Managing Unknown Records"
source: "/space/nios90/1422688449"
pageId: "1422688449"
---
An unknown record is a resource record for which the record type is unknown to NIOS. You can create an unknown record and assign a type to it. NIOS converts the unknown record to the type you assign. You can assign a type listed at [https://www.iana.org/assignments/dns-parameters/dns-parameters.xhtml](https://www.iana.org/assignments/dns-parameters/dns-parameters.xhtml) . For information about types that you cannot assign, see the *Guidelines for Creating Unknown Records* section.
