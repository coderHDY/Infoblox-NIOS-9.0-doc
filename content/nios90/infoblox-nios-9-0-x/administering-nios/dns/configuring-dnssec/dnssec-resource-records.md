Following are the DNSSEC RR types:

- DNS Public Key (DNSKEY) resource records — For information, see [*DNSKEY Resource Records*](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dnssec/dnskey-resource-records).
- Resource Record Signature (RRSIG) records — For information, see [*RRSIG Resource Records*](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dnssec/rrsig-resource-records).
- Next Secure (NSEC/NSEC3) records — For information, see [*NSEC/NSEC3 Resource Records*](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dnssec/nsec-nsec3-resource-records).
- NSEC3PARAM records — For information, see [*NSEC3PARAM Resource Records*](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dnssec/nsec3param-resource-records).
- Delegation Signer (DS) resource records — For information, see [*DS Resource Records*](/nios90/infoblox-nios-9-0-x/administering-nios/dns/configuring-dnssec/ds-resource-records).

For detailed information about each RR, refer to *RFC* *4034,* *Resource* *Records* *for* *the* *DNS* *Security* *Extensions* and *RFC* *5155,* *DNS* *Security* *(DNSSEC)* *Hashed* *Authenticated* *Denial* *of* *Existence.*

> **warning**
>
> ### Note
> 
> The appliance supports IDNs for DNSKEY records, DS records, NSEC records, NSEC3PARAM records, and RRSIG records.
