---
To add a stub zone, you must identify the Infoblox appliance that hosts the stub zone, and provide the IP address of the primary server.

You can also add stub zones for Microsoft servers that are managed by Grid members. For information, see [*Managing Microsoft Windows Servers*](/nios90/infoblox-nios-9-0-x/administering-nios/configuring-microsoft-windows-servers/managing-microsoft-windows-servers/about-managing-microsoft-windows-servers).

You can configure a stub zone for forward mapping or reverse mapping zones.
