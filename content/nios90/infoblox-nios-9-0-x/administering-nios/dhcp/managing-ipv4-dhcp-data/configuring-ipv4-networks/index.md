When you create an IPv4 network, you can do so from scratch or from a network template that contains predefined properties. When you use a template to create a network, the properties of the template apply to the new network. For information about network templates, see [*About IPv4 DHCP Templates*](/nios90/infoblox-nios-9-0-x/administering-nios/dhcp/managing-dhcp-templates/about-ipv4-dhcp-templates). You can also create an IPv4 network from the Tasks Dashboard, as described in [*Tasks Dashboard*](/nios90/infoblox-nios-9-0-x/using-the-grid-manager-interface/dashboards/tasks-dashboard).  
  
After you create IPv4 networks, you can combine them into shared networks or create ranges and fixed addresses.
