To delete a roaming host:

1. From the **Data** **Management** tab, select the **DHCP** tab -&gt; **Networks** tab -&gt; **Roaming** **Hosts** -&gt; *roaming_host* checkbox, and then select **Delete** or **Schedule** **Delete** from the drop-down menu.
2. To delete the roaming host now, in the *Delete* *Confirmation* dialog box, click **Yes**. To schedule the deletion, see [*Scheduling Deletions*](/nios90/infoblox-nios-9-0-x/using-the-grid-manager-interface/scheduling-new-ipam-dhcp-objects-and-associated-/scheduling-deletions). The Grid Manager puts the deleted roaming host in the Recycle Bin, if enabled.
