To start the DHCP service on the Grid Master:

1. From the **Data** **Management** tab, select the **DHCP** tab -&gt; **Members** tab.
2. Select the Grid Master [gm.campus1.school.edu](http://gm.campus1.school.edu), and the two members, [ds1.campus2.school.edu](http://ds1.campus2.school.edu) and [ds2.campus2.school.edu](http://ds2.campus2.school.edu).
3. Expand the Toolbar and click **Start**.
4. In the *Start* *Member* *DHCP* *Service* dialog box, click **Yes**.
5. Grid Manager starts DHCP services on the Grid Master and on the selected members.
