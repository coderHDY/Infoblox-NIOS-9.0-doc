You can view a list of all IPv4 and IPv6 DHCP templates and delete the templates that are not required.

# Viewing Templates

To view a list of all IPv4 and IPv6 DHCP templates:

1. From the **Data** **Management** tab, select the **DHCP** tab -&gt; **Templates** tab.
2. Grid Manager displays the following information:
   
   - **Name**: The name of the template.
   - **Type**: The template type, such as **IPv4** **Network** **Template** or **IPv6 Network Template**.
   - **Comment**: The information you entered about the template.
   - **Site**: The site to which the template belongs. This is one of the predefined extensible attributes.

You can select predefined and user defined extensible attributes for display. You can also do the following in this panel:

- Modify some of the data in the table. Double click a row of data, and either edit the data in the field or select an item from a drop-down list. Note that some fields are read-only. For more information about this feature, see [*Modifying Data in Tables*](/nios90/infoblox-nios-9-0-x/using-the-grid-manager-interface/about-the-grid-manager-interface/About+the+Grid+Manager+Interface#%5BinlineExtension%5DModifying-Data-in-Tables).
- Sort the displayed data in ascending or descending order by column.
- Delete a selected template or multiple templates. For information, see [*Deleting Templates *](/nios90/infoblox-nios-9-0-x/administering-nios/dhcp/managing-dhcp-templates/viewing-and-deleting-templates/Viewing+and+Deleting+Templates#Deleting-Templates)below.
- Use filters and the **Go to** function to narrow down the list. With the autocomplete feature, you can just enter the first few characters of an object name in the **Go to** field and select the object from the possible matches.
- Create a quick filter to save frequently used filter criteria. For information, see [*Using Quick Filters*](/nios90/infoblox-nios-9-0-x/using-the-grid-manager-interface/performing-basic-operations-using-grid-manager/finding-and-restoring-data/Finding+and+Restoring+Data#%5BinlineExtension%5DUsing-Quick-Filters).
- Select an object and edit its information.
- Print or export the data in the panel.

# Deleting Templates

To delete a template:

1. From the **Data** **Management** tab, select the **DHCP** tab -&gt; **Templates** tab -&gt; *template* checkbox, and then click the Delete icon.
2. In the *Delete* *Confirmation* dialog box, click **Yes**.
