To download the syslog file to a specified directory, if you want to analyze it later, complete the following:

1. From the **Administration** tab, select the **Logs** tab -&gt; **Syslog** tab, and then click the Download icon.
2. Navigate to a directory where you want to save the file, optionally change the file name (the default names are *node_1_sysLog.tar.gz* and *node_2_sysLog.tar.gz*), and then click **OK**. If you want to download multiple syslog files to the same location, rename each downloaded file before downloading the next.

> **warning**
>
> Note
> 
> If your browser has a pop-up blocker enabled, you must turn off the pop-up blocker or configure your browser to allow pop-ups for downloading files.
