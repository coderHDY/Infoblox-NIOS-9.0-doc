---
title: "Modifying SNMPv3 Users"
source: "/space/nios90/1406435330"
pageId: "1406435330"
---
1. From the **Administration** tab, select the **SNMPv3** **Users** tab -&gt; *snmpv3user*, and then click the Edit icon.
2. The *SNMPv3* *User* editor provides the following tabs from which you can edit data:
   
   - **General**: Modify the data as described in *Configuring* *SNMPv3* *Users* in this topic.
   - **Extensible** **Attributes**: Add and delete extensible attributes that are associated with the SNMPv3 user account. You can also modify the values of extensible attributes. For information, see [*Using Extensible Attributes*](/nios90/infoblox-nios-9-0-x/administering-nios/appliance-administration/managing-appliance-operations/managing-extensible-attributes)*.*
3. Save the configuration.
