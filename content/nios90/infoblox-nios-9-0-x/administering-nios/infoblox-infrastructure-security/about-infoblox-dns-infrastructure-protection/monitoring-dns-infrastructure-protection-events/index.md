- Syslog, as described in *[Monitoring through Syslog](/nios90/by-id/280275770)*.
- Status Dashboard, as described in *[DNS Infrastructure Protection Statistics Widget](/nios90/by-id/280668746)*.
- DNS Infrastructure Protection report, as described in *[DNS Infrastructure Protection Reports](/nios90/by-id/280408700)*.
