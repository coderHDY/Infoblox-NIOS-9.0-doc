title: "Modifying IPv4 and IPv6 Network Containers and Networks"
source: "/space/nios90/1346274199"
pageId: "1346274199"
---
You can modify existing network settings, with the exception of the network address and subnet mask. To modify an IPv4 or IPv6 network container or network:

1. From the **Data** **Management** tab, select the **IPAM** tab -&gt; *network* checkbox, and then click the Edit icon.
2. In the *DHCP* *Network* editor, modify the network settings as described in [*Modifying IPv4 Networks*](/nios90/infoblox-nios-9-0-x/administering-nios/ip-address-management/managing-ip-addresses/managing-ipv4-networks) or [*Modifying IPv6 Networks*](/nios90/infoblox-nios-9-0-x/administering-nios/ip-address-management/managing-ip-addresses/managing-ipv6-networks).
