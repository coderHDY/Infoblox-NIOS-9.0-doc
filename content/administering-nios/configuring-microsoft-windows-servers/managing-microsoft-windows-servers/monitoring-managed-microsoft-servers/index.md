title: "Monitoring Managed Microsoft Servers"
source: "/space/nios90/280761116"

---
You can monitor the status of managed Microsoft servers from the Dashboard and from various panels in the **Infoblox** **Grid** tab. Grid Manager also maintains a log for each managed Microsoft server. You can monitor Microsoft servers and their services as follows:

- You can view the *Microsoft* *Servers* *Status* widget on the Dashboard. For information, see [*Microsoft Servers Status Widget*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/537002114).
- You can view the status of Microsoft servers. For information, see Viewing DNS and DHCP Service Status on Microsoft Servers below.
- You can view the logs of the Microsoft servers. For information, see Viewing Synchronization Logs below.
