Static (member) licenses are member-specific licenses that are mapped to the hardware IDs (also called as serial numbers) of individual NIOS or vNIOS appliances. Static licenses cannot be transferred among appliances. They come pre-installed on physical NIOS appliances according to the software packages you order at the time of purchase.  
Static licenses can be perpetual or non-perpetual. A perpetual license does not have an expiration date. When you install a new static license with an expiration date beyond the existing license, it replaces the existing license. After you install static licenses on an appliance, you can view their status in Grid Manager.
