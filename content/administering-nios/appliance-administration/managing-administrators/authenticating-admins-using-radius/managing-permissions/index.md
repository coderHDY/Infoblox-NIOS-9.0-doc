After you define permissions for an admin group and role, you can do the following:

- View the permissions, as described in [*Viewing Permissions*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1343095544).
- Filter a list of permissions, as described in [*Filtering the List of Permissions*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1343324687).
- Modify the permissions, as described in [*Modifying Permissions*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1343324720).
- Delete the permission, as described in [*Deleting Permissions*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1343324756).
