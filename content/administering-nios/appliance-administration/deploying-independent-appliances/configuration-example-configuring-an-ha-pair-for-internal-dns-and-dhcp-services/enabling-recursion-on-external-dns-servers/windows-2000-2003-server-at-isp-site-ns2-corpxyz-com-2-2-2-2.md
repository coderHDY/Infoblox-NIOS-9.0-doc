1. Click **Start** -> **All Programs** -> **Administrative Tools** -> **DNS**.
2. Right-click **ns3**, and then select **Properties** -> **Advanced**.
3. On the *Advanced* page in the *ns3* *Properties* dialog box, clear the **Disable recursion** checkbox.
