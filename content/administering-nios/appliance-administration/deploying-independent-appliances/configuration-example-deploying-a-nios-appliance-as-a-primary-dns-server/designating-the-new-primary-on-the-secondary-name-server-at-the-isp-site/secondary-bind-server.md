1. Open the named.conf file using a text editor and set ns1 (with NAT address 1.1.1.2) as the primary from which ns2 receives zone transfers in the named.conf file for the [corpxyz.com](http://corpxyz.com) zone.
2. After editing the named.conf file, restart DNS service for the change to take effect.
