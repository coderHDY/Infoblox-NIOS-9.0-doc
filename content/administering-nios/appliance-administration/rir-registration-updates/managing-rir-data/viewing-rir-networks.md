You can view a list of IPv4 and IPv6 RIR networks in the **Data** **Management** tab -> **IPAM** **tab** or the **Data** **Management** tab -> **DHCP** tab -> **Networks** tab -> **Networks** section. For more information, see [*IPAM Home*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1345162224) and [*Viewing Networks*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1432715867).
