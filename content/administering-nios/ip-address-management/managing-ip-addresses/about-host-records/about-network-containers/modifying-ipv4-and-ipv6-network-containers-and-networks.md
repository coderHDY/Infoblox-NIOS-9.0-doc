---
title: "Modifying IPv4 and IPv6 Network Containers and Networks"
source: "/space/nios90/1346274199"
pageId: "1346274199"
---
You can modify existing network settings, with the exception of the network address and subnet mask. To modify an IPv4 or IPv6 network container or network:

1. From the **Data** **Management** tab, select the **IPAM** tab -> *network* checkbox, and then click the Edit icon.
2. In the *DHCP* *Network* editor, modify the network settings as described in [*Modifying IPv4 Networks*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666698) or [*Modifying IPv6 Networks*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764556).
