You can do the following from the IP Map and List panels:

- Add IP addresses to existing hosts. For information, see Adding IP Addresses to Existing Host Records below.
- Clear unmanaged IP addresses. For information, see Clearing Unmanaged Data see below.
- Convert objects to other object types. For information, see [*Converting*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Objects*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Associated*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*with*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IP*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[.](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531/Managing+IPv4+and+IPv6+Addresses#bookmark1286)
- Reclaim IP addresses. For information, see [*Reclaiming*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Objects*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Associated*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*with*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IPv4*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*and*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IPv6*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[.](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531/Managing+IPv4+and+IPv6+Addresses#bookmark1293)
- Ping IP addresses. For information, see [*Pinging*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IP*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531).

For information, see [*IP*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670)[*Discovery*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670)[*and*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670)[*vDiscovery*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280273670).

For information, see [*Resolving*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280406620)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280406620)[*Conflicting*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280406620)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280406620)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280406620).
