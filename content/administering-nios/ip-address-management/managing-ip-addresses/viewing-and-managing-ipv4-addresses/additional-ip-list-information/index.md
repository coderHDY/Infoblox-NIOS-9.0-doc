You can display all available extensible attributes. You can also sort the list of IP addresses in ascending or descending order by **IP** **Address** only. If you enabled the IP Discovery feature, you can configure the IP List panel to display discovered data and fields imported from NetMRI appliances. For information about integrating discovered data from NetMRI, see [*Integrating*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[*Discovered*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[*Data*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[*From*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361)[*NetMRI*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764361).  
You can select an IP address from the List panel and view the following information about it:

- General information, as described in [*IP*](/wiki/pages/resumedraft.action?draftId=26772634)[ ](/wiki/pages/resumedraft.action?draftId=26772634)[*Address*](/wiki/pages/resumedraft.action?draftId=26772634)[ ](/wiki/pages/resumedraft.action?draftId=26772634)[*Header*](/wiki/pages/resumedraft.action?draftId=26772634)[ ](/wiki/pages/resumedraft.action?draftId=26772634)[*Panel*](/wiki/pages/resumedraft.action?draftId=26772634).
- Data retrieved through a network discovery or integrated from a PortIQ appliance, as described in [*Viewing*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666433)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666433)[*Discovered*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666433)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666433)[*Data*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666433).
- The records associated with the IP address, as described in [*Related*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[*Objects*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[.](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)
- Audit history, as described in [*Audit*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[*History*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533).
- Detailed lease information, as described in [*Viewing*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976)[*Detailed*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976)[*Lease*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976)[*Information*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662976).

You can also do the following from the IP List panel:

- Click **Go** **to** **DHCP** **View** to view DHCP properties of a selected network. For information, see [*Modifying*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280761742)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280761742)[*IPv4*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280761742)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280761742)[*Networks*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280761742).
- Click the Ping icon to ping a selected IP address. For information, see [*Pinging*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IP*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531).
