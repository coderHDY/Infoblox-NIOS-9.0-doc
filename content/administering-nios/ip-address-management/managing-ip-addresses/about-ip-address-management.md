IPAM is the allocation, administration, reporting, and tracking of public and private IP spaces, network devices, and their associated data. It comprises the deployment of DNS and DHCP services and the monitoring of network devices and performance to ensure data integrity and security of your networks.

The Infoblox IPAM implementation is a feature-rich and easy-to-use solution that encompasses support for IPv4, IPv6, network discovery, and automated monitoring. Infoblox IPAM provides tools that integrate the allocation, administration, reporting, and tracking of your entire network space. The below figure highlights the Infoblox IPAM implementation.

You can perform the following IPAM tasks to effectively manage and control your network:

- Create host records. Host records integrate the DNS records and DHCP data of a network device. You can use a host record to manage a network device from one central point. For more information, see [*About Host Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666779)[.](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666779)
- Create IPv4 and IPv6 networks. For information, see [*About Network Containers*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666779).
- Discover devices in IPv4 and IPv6 networks and other Objects created in IPAM. For more information about Adding IPv4 and IPv6 Network Containers and Networks and Adding Host Records, see [*About Host Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666779). View your IPv4 network and address utilization in a graphical mode. For information, see [*IPv4 Network Map*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666698)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666698/Managing+IPv4+Networks#IPNetworkMap)and [*IP Map*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616).
- When necessary, resize, join, or split networks. For information about Resizing IPv4 Networks, Splitting IPv4 Networks into Subnets, Joining IPv4 Networks see*,** *[*Managing IPv4 Networks*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666698)*.* For information about Splitting IPv6 Networks into Subnets, and Joining IPv6 Networks, see [*Managing IPv6 Networks*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/319292098).
- Manage IPv4 and IPv6 address data. For information, see [*Viewing*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[*and*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[*Managing*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[*IPv4*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616)[,](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666616) [*Viewing*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[*IPv6*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[*Data*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280764533), and [*Managing*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IPv4*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*and*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*IPv6*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[ ](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531)[*Addresses*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280666531).
