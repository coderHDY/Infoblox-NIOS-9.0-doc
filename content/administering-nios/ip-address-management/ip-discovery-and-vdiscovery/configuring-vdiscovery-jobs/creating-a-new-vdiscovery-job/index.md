To create a new vDiscovery job, complete the following tasks:

1. Name the new job and select a member to perform the vDiscovery, as described in the [*Selecting the vDiscovery Member*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1468499160/Selecting+the+vDiscovery+Member?atlOrigin=eyJpIjoiOWNlNjM2MjM3ZGE3NDgzYmJkMmFkNzU0MzYxNWQ1MTUiLCJwIjoiYyJ9) section. Do not use control characters or undefined Unicode characters in your job name.
2. Select a cloud platform for the vDiscovery, as described in the [Selecting the Endpoint Server](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1468630069/Selecting+the+Endpoint+Server?atlOrigin=eyJpIjoiZjdmOGE4YjI0MGE5NDlkMGFhN2QyYjIwZmQwZDYzMDAiLCJwIjoiYyJ9) section.
3. Define network views for public and private IP addresses, as described in the [*Defining Network Views*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1468335066/Defining+Network+Views?atlOrigin=eyJpIjoiMzMwNmVmMDg5OTI4NGQ4MmJjNzNiODk3OTFjMGZiNGIiLCJwIjoiYyJ9) section.
4. Define policies for handling discovered data, as described in the [Defining Policies for Handling Discovered Data](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1468466176/Defining+Policies+for+Handling+Discovered+Data?atlOrigin=eyJpIjoiN2U1ZDQyZTk4ZjM3NDE0MDhmZmZjYTIxNDZiMWRkMzkiLCJwIjoiYyJ9) section[.](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/704348239/Configuring+vDiscovery+Jobs#bookmark1347)
5. Optionally, you can enable NIOS to automatically create DNS records for newly discovered VMs using their IP addresses, as described in the [Creating DNS Records for Newly Discovered VMs](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1468466214/Creating+DNS+Records+for+Newly+Discovered+VMs?atlOrigin=eyJpIjoiNDg5MzRhNjA3ZjJmNGNiNGI4OTZjY2RmMTAyYTVmMmYiLCJwIjoiYyJ9) section below.
6. Schedule the vDiscovery job, as described in the in [*Scheduling vDiscovery Jobs*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1468335102/Scheduling+vDiscovery+Jobs?atlOrigin=eyJpIjoiYTFiYWQ3ZGJlYTA5NGY3NDllMzcwOTgyOGE0MDIxMzUiLCJwIjoiYyJ9) section below.
