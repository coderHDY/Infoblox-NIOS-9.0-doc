When you create an IPv4 network, you can do so from scratch or from a network template that contains predefined properties. When you use a template to create a network, the properties of the template apply to the new network. For information about network templates, see [*About IPv4 DHCP Templates*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280404118). You can also create an IPv4 network from the Tasks Dashboard, as described in [*Tasks Dashboard*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280759015).  
  
After you create IPv4 networks, you can combine them into shared networks or create ranges and fixed addresses.
