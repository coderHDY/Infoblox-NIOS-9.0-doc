---
You can assign a forwarding member NS group to a forward zone when you first create it using the *Add* *Forward* *Zone* wizard and when you edit an existing forward zone using the *Forward* *Zone* editor. For information about creating a forward zone using the wizard, see [*Configuring a Forward Zone*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280762909).  
Complete the following to assign a forwarding member NS group to a forward zone:

1. From the **Data** **Management** tab -> **DNS** tab -> **Zones** tab -> *forward* *zone* checkbox, and then click the Edit icon.
2. In the *Forward* *Zone* editor, click **Name** **Servers**.
3. Select **Use** **this** **name** **server** **group**, and then select the forwarding member NS group from the drop-down list.
