Following are the DNSSEC RR types:

- DNS Public Key (DNSKEY) resource records — For information, see [*DNSKEY Resource Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280762383).
- Resource Record Signature (RRSIG) records — For information, see [*RRSIG Resource Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280664613).
- Next Secure (NSEC/NSEC3) records — For information, see [*NSEC/NSEC3 Resource Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280664563).
- NSEC3PARAM records — For information, see [*NSEC3PARAM Resource Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280405045).
- Delegation Signer (DS) resource records — For information, see [*DS Resource Records*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280271884).

For detailed information about each RR, refer to *RFC* *4034,* *Resource* *Records* *for* *the* *DNS* *Security* *Extensions* and *RFC* *5155,* *DNS* *Security* *(DNSSEC)* *Hashed* *Authenticated* *Denial* *of* *Existence.*

> **warning**
>
> ### Note
> 
> The appliance supports IDNs for DNSKEY records, DS records, NSEC records, NSEC3PARAM records, and RRSIG records.
