To enable NXDOMAIN redirection and configure its properties:

1. Configure NXDOMAIN rulesets. You can create NXDOMAIN rulesets through Grid Manager, as described in  Creating Rulesets below. You can also specify the rulesets in a CSV file and import the file to the Grid, as described in [*Importing and Exporting Data using CSV Import*](https://infoblox-docs.atlassian.net/wiki/pages/createpage.action?spaceKey=nios90&title=Importing%20and%20Exporting%20Data%20using%20%20CSV%20Import&linkCreation=true&fromPageId=1420198194)*.*
2. Enable this feature and specify the redirection IP addresses, as described in  Enabling NXDOMAIN Redirection below.
