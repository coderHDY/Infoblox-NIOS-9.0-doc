After you have set up reporting members and defined clustering type, you can monitor the cluster status through the following:

- View the reporting member service status, as described in Monitoring Grid Services, see [*Monitoring Services*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280662651).
- Check license usage by the reporting member, as described in [*Home Dashboards*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280401011).
