To enable and start using the Infoblox reporting solution on supported reporting appliances, complete the following:

1. Install a valid Reporting license on the reporting appliance, as described in *[Licensing Requirements](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1458241996)*. Note that Infoblox now offer subscription licenses for the reporting solution.
2. Set administrative permissions, as described in *[Administrative Permissions for Reporting and Analytics](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280857269)*.
3. Configure reporting properties, as described in *[Grid Reporting Properties](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280267688)*.
4. Start the reporting service on the indexer and forwarders.
