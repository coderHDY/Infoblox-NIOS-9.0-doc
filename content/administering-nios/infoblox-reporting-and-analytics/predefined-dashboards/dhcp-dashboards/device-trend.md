The *Device* *Trend* dashboard provides trends for the top operating systems used by remote clients in a given time frame. The default dashboard displays line graphs for the top 10 operating systems used by remote clients over the last 24 hours. Each of the operating system is represented with a different color line graph. For more information about DHCP fingerprint detection, see [*DHCP Fingerprint Detection*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280402989).
