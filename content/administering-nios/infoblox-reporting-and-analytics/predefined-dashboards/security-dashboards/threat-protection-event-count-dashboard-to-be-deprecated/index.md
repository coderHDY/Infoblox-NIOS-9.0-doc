> This dashboard will be deprecated from NIOS 9.1.0 onwards. For future reference, use [*DNS Infrastructure Protection Event Count Dashboard*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1428947890).

The *Threat Protection Event Count Dashboard* is a predefined custom dashboard comprising sub-dashboards listed in the table below. Duplicates of these sub-dashboards exist under new names, replacing "Threat Protection" with "DNS Infrastructure Protection" since NIOS 9.0.8. Dashboards with the previous names will be deprecated from 9.1.0 onwards.

| [*Threat Protection Event Count By Member Trend*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90draft/pages/1811972236) | [*DNS Infrastructure Protection Event Count By Member Trend*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1429242979) |

| [*Threat Protection Event Count By Time*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1861386253) | [*DNS Infrastructure Protection Event Count By Time*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1429013704) |
