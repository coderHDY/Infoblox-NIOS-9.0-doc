The *Threat Protection* *Event* *Count* *By* *Member* *Trend* sub-dashboard provides event count trends on members that supports Threat Protection in a given time frame. This dashboard tracks events on a member over a given time frame. By default, this sub-dashboard displays a line chart that shows events trends over the last day on the selected member and also displays the top 5 appliances in descending order.
