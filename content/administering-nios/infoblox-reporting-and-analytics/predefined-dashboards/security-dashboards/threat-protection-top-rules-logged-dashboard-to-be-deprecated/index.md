> **Note**
> 
> This dashboard will be deprecated from NIOS 9.1.0 onwards. For future reference, use [*DNS Infrastructure Protection Top Rules Logged Dashboard*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1428947920).

The *Threat Protection Top Rules Logged Dashboard* is a predefined custom dashboard comprising sub-dashboards listed in the table below. Duplicates of these sub-dashboards exist under new names, replacing "Threat Protection" with "DNS Infrastructure Protection" since NIOS 9.0.8. Dashboards with the previous names will be deprecated from 9.1.0 onwards.

| **Sub-Dashboard with**   **Previous Name** | **Sub-Dashboard with**   **Changed Name** |
| --- | --- |
| [Threat Protection Top Rules Logged](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1856045627) | [*DNS Infrastructure Protection Top Rules Logged*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1429242918) |
| [Threat Protection Top Rules Logged by Source](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1858241115) | [*DNS Infrastructure Protection Top Rules Logged by Source*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/1428849707) |
