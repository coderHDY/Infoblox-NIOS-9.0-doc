You can configure a Cisco ISE endpoint by using the **Ecosystem** > **Outbound** **Endpoint** tab. The instructions in this section pertain to configuring Cisco ISE Outbound Endpoint 2.6, 2.7, 3.0, and 3.1 using the **Outbound** **Endpoint** tab.

You can configure Cisco ISE 3.1 only using the outbound endpoint.

To configure the Cisco ISE endpoint using the **Outbound** **Endpoint** tab, as described in Configuring Cisco ISE Endpoints Using Outbound Endpoint, see [*Cisco ISE Integration*](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/330104909).
