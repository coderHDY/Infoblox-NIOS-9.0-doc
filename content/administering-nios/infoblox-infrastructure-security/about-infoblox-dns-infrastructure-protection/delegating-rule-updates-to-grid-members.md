title: "Delegating Rule Updates to Grid Members"
source: "/space/nios90/280270029"

---
You can delegate ruleset updates to other Grid members if desired. You can also configure the interface you want to use for automatic updates on the members. For information about how to configure the Members and Interfaces for Automatic Updates, see [https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280765075](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280765075).
