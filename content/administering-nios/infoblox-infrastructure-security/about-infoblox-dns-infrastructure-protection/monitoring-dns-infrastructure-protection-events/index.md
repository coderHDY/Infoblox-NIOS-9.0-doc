- Syslog, as described in *[Monitoring through Syslog](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280275770)*.
- Status Dashboard, as described in *[DNS Infrastructure Protection Statistics Widget](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280668746)*.
- DNS Infrastructure Protection report, as described in *[DNS Infrastructure Protection Reports](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280408700)*.
