The IB-DOTQUERYRATE-MIB contains one object, ibDOTQueryRate, which provides information about the DoT queries per second. Note that the MIB variable (ibDataplaneOne) for DoT query rate has an OID of 1.3.6.1.4.1.7779.3.1.1.5.1.1.0 where 1.3.6.1.4.1.7779 is the prefix.
