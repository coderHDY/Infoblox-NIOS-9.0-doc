The IB-DOHQUERYRATE-MIB contains one object, ibDOHQueryRate, which provides information about the DoH queries per second. Note that the MIB variable (ibDataplaneOne) for DoH query rate has an OID of 1.3.6.1.4.1.7779.3.1.1.5.1.2.0 where 1.3.6.1.4.1.7779 is the prefix.
