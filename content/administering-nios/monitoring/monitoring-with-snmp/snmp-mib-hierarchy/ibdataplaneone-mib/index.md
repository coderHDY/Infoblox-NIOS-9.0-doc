The ibDataplaneOne MIB provides information about queries per second for DNS over TLS and DNS over HTTPS. It includes two modules, IB-DOTQUERYRATE-MIB for DNS over TLS query rate data and IB-DOHQUERYRATE-MIB for DNS over HTTPS query rate data.
