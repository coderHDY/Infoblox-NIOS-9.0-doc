Grid Manager provides tools for monitoring the status of the Grid, members, and services. You can monitor overall Grid and member status from the Dashboard, which provides a high-level view of your Grid, members and IP address data, and easy access to tasks. For information, see *[Dashboards](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280266914)*.

Grid Manager also displays status icons to indicate the state of appliances, services, database capacity, Ethernet ports, HA, and Grid replication. Depending on your appliance, Grid Manager can display status icons for the power supplies as well as icons to indicate the state of the RAID array and disk controller backup battery.  
You can monitor detailed status of the Grid, members, and services, and then decide how to manage them. Note that when any member or service encounters issues, the appliance sends SNMP traps. For information, see *[Monitoring with SNMP](https://infoblox-docs.atlassian.net/wiki/spaces/nios90/pages/280401198)*.
